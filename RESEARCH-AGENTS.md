# 임팩트클라이밋 기후테크 리서치 에이전트 시스템

## 개요

카카오임팩트 임팩트클라이밋 사업의 기후테크 생태계 파악을 위한 6개 리서치 에이전트 체계.
각 에이전트는 독립적으로 실행 가능하며, 수집된 지식은 `knowledge-base/` 에 구조화되어 축적됩니다.

---

## 에이전트 구성

### Agent 1: 기후테크 투자 트래커 (`/invest`)
> 글로벌/국내 기후테크 투자 동향, VC 활동, 펀드 조성 추적

**수집 대상:**
- 글로벌 기후테크 VC 투자 동향 (BloombergNEF, PitchBook, CTVC, Sightline Climate)
- 국내 기후테크 펀드 조성 현황 (미래환경산업투자펀드 810억 등)
- 주요 딜 (시리즈별 투자, M&A, IPO)
- 분기별 투자 트렌드 변화

**산출물:** `knowledge-base/01-investment/` 에 분기/월별 리포트
**활용:** 서밋 딥다이브 세션 기획, 투자자 연사 발굴, 스타트업 피칭 대상 선정

---

### Agent 2: 기후테크 스타트업 스카우터 (`/startups`)
> 국내외 기후테크 스타트업 발굴, 기술 분류, 성장 단계 추적

**수집 대상:**
- 분야별 스타트업 매핑 (에너지, 모빌리티, 농식품, CCUS, 순환경제, 생물다양성, 기후데이터)
- 국내 기후테크 스타트업 DB (신규 창업, 투자 유치, 피봇)
- 해외 주목 스타트업 (특히 아시아-태평양)
- 기후 AI 스타트업 전문 트래킹

**산출물:** `knowledge-base/02-startups/` 에 스타트업 프로필 카드, 분야별 맵
**활용:** 서밋 피칭 스테이지 후보, Climate Pod 컨텐츠, 네트워크 확장

---

### Agent 3: 기후 정책 모니터 (`/policy`)
> 글로벌/국내 기후 정책, 규제 변화, 제도 동향 추적

**수집 대상:**
- 국내: 기후에너지환경부 정책, 탄소중립 로드맵, NDC, 배출권거래제, RE100
- 글로벌: EU CBAM, 미국 IRA/트럼프 정책 변화, COP31 의제
- 아시아: 중국/일본/동남아 기후 정책 동향
- 기후테크 스타트업 관련 규제/지원 정책

**산출물:** `knowledge-base/03-policy/` 에 정책 브리핑 노트
**활용:** 서밋 인사이트 세션 기획, 정책 연사 발굴, 정부 협력 포인트 도출

---

### Agent 4: AI x Climate 기술 인텔리전스 (`/ai-climate`)
> AI가 기후문제에 적용되는 최신 기술 동향 및 사례 분석

**수집 대상:**
- AI 기반 기후 솔루션 (예측 모델, 최적화, 모니터링, MRV)
- 빅테크의 기후 AI 투자 (Google, MS, Meta, 카카오 등)
- AI 인프라의 에너지 이슈 (데이터센터 전력, 그린 AI)
- 학술 연구 동향 (Climate AI 논문, 벤치마크)
- 바이브코딩/AI Agent 활용 기후 솔루션 사례

**산출물:** `knowledge-base/04-ai-climate/` 에 기술 트렌드 리포트
**활용:** 서밋 핵심 주제 설정, AI Agent 플랫폼 기획, 데모 세션 구성

---

### Agent 5: 기후테크 생태계 네트워크 매퍼 (`/ecosystem`)
> 주요 기관, 인물, 프로그램, 연결 구조 파악

**수집 대상:**
- 국내 기후테크 생태계 플레이어 (액셀러레이터, VC, 정부기관, 대기업 CSR/CVC)
- 글로벌 기후 네트워크 (Breakthrough Energy, SOSV, Elemental Excelerator 등)
- 주요 인물 프로필 및 동향
- 대학/연구소 기후테크 센터

**산출물:** `knowledge-base/05-ecosystem/` 에 기관/인물 프로필, 생태계 맵
**활용:** 서밋 초청자 명단 확대, 협력 기관 발굴, MOU 후보 식별

---

### Agent 6: 글로벌 기후 이벤트 인텔리전스 (`/events`)
> 주요 글로벌 기후 서밋, 컨퍼런스, 보고서 발간 동향

**수집 대상:**
- 글로벌 기후 컨퍼런스 (COP, Climate Week NYC, BNEF Summit, Singapore GGEF 등)
- 주요 보고서 발간 (IPCC, IEA, BloombergNEF, McKinsey 등)
- 아시아-태평양 기후테크 이벤트
- 경쟁/벤치마크 서밋 분석

**산출물:** `knowledge-base/06-events/` 에 이벤트 캘린더, 주요 보고서 요약
**활용:** 서밋 일정 최적화, 글로벌 의제 선점, 연사/참가자 동선 파악

## 실행 방법

### 1. 개별 에이전트 실행
```
"Agent 1 실행해줘" 또는 "/invest 2026 Q2 동향 조사해줘"
```
특정 에이전트를 지정하여 해당 영역의 최신 정보를 수집합니다.

### 2. 전체 스캔 (Full Sweep)
```
"전체 리서치 스캔 돌려줘"
```
6개 에이전트를 병렬로 실행하여 전 영역의 최신 동향을 한 번에 수집합니다.

### 3. 서밋 기획 모드
```
"2026 서밋 기획용 리서치 해줘"
```
서밋 기획에 필요한 주제/연사/스타트업/정책 포인트를 종합적으로 도출합니다.

### 4. 정기 업데이트
```
"월간 기후테크 업데이트 해줘"
```
각 에이전트의 최근 1개월 변화를 요약하는 월간 브리핑을 생성합니다.
